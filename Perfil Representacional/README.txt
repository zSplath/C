Projeto da disciplina de Lógica de Programação no IFSP

Esse programa serve para fazer um teste de perfil representacional,
descobrir se você é: Auditivo, Digital, Cinestésico ou Visual.

Por requisito do projeto, 
Primeiro é necessário logar com usuário "MASTER" e cadastrar o questionário.
Após isso, pode-se abrir o programa novamente e realizar o teste.

Sumário de arquivos:
main.c -> código do projeto
Projeto.exe -> executável
